Mechanical Keyboard FN Function and Lighting Operation Instructions

Basic FN Functions:

  • FN+\ Switch light mode
  • FN+DEL Switch light color
  • FN+ü°ë/ü°ì Light brightness adjustment
  • FN+ü°ê/ü°í Adjust the speed of light breathing or changes
  • FN+Tab One key keeps the white light on, then press to exit
  • FN+RCtrl Switch keyboard side light mode
  • FN+END Switch function light bar mode
  • FN+Backspace One click turn off light
  • FN+ESC Light Reset
  • FN+Win Lock Win
